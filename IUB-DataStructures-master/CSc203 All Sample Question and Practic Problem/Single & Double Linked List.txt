Single and Double Linked List:
 
1.Search a node and delete it.

2.Search a node and insert another node after the searching node.

3.Search a node and insert another node before the searching node.

4.Search a node and insert another node after and before the searching node.

5.Search a node in the list and for each occurance delete the node and enter a new node in the same place.

6.Search a node and in second,Third,Fourth occurance Insert a number before, after the search node.

7.Search a node and if it is found then add one node before this node and also add one node after the node for all the occurance.

8.Search a node and delete all the occurance.
